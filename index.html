    eBird API Documentation

    Introduction

    The eBird API provides access to a vast collection of bird observation data worldwide. It allows you to retrieve the latest bird sightings, search for species in specific areas, and explore historical birding records.

    üê¶ GET Recent Observations in a Region

    Retrieve the latest bird observations (from the past 30 days) for a given country, state, county, or location. The results include only the most recent sighting for each species within the specified area.

    üîó Endpoint

    API Endpoint

    Base URL: https://api.ebird.org
    Endpoint: /v2/data/obs/{regionCode}/recent
    Method: GET
    Authentication: API token required (X-eBirdApiToken)

    üõ†Ô∏è URL Parameters

    Parameter Description
    regionCode The country, subnational1, subnational2, or location code.

    üìå Query Parameters

    Parameter Values Default Description
    back 1-30 14 Number of days back to fetch observations.
    cat Any available category (all) Only fetch observations from specified taxonomic categories.
    hotspot true, false false Fetch only observations from hotspots.
    includeProvisional true, false false Include unreviewed observations.
    maxResults 1-10000 (all) Limit the number of observations returned.
    r Any location code (none) Fetch observations from up to 10 locations.
    sppLocale Any available locale en Language for species common names.

    üíª Example Requests

    curl --location 'https://api.ebird.org/v2/data/obs/KZ/recent' \
--header 'X-eBirdApiToken: {{x-ebirdapitoken}}'

    import requests

url = "https://api.ebird.org/v2/data/obs/KZ/recent"
headers = {"X-eBirdApiToken": "{{x-ebirdapitoken}}"}

response = requests.get(url, headers=headers)
print(response.json())

    fetch("https://api.ebird.org/v2/data/obs/KZ/recent", {
    method: "GET",
    headers: { "X-eBirdApiToken": "{{x-ebirdapitoken}}" }
})
.then(response => response.json())
.then(data => console.log(data));

    üê¶ GET Recent nearby observations

    Get the latest bird species observations from the past 30 days within a 50-kilometer radius of the provided coordinates. The results show only the most recent sighting of each species in the specified area.

    üîó Endpoint

    API Endpoint

    Base URL: https://api.ebird.org
    Endpoint: /v2/data/obs/geo/recent?lat={{lat}}&lng={{lng}}
    Method: GET
    Authentication: API token required (X-eBirdApiToken)

    üõ†Ô∏è URL Parameters

    Parameter Description
    lat The latitude of the central point for the observation search.
    lng The longitude of the central point for the observation search.

    üìå Query Parameters

    Parameter Values Default Description
    back 1-30 14 The number of days back to fetch observations.
    cat Any available category (all) Only fetch observations from these taxonomic categories.
    dist 0-50 25 The search radius from the given position, in kilometers.
    hotspot true, false false Only fetch observations from hotspots.
    includeProvisional true, false false Include observations which have not yet been reviewed.
    lat -90 to 90 ‚Äî Required. Latitude to 2 decimal places.
    lng -180 to 180 ‚Äî Required. Longitude to 2 decimal places.
    maxResults 1-10,000 (all) Only fetch this number of observations.
    sort date, species date Sort observations by taxonomy or by date, most recent first.
    sppLocale Any available locale en Use this language for species common names.

    üíª Example Requests

    curl --location 'https://api.ebird.org/v2/data/obs/geo/recent?lat={{lat}}&lng={{lng}}&dist={{dist}}&back={{back}}&maxResults={{maxResults}}&sort={{sort}}&hotspot={{hotspot}}&includeProvisional={{includeProvisional}}&sppLocale={{sppLocale}}' \
--header 'X-eBirdApiToken: {{x-ebirdapitoken}}'

    import requests

url = "https://api.ebird.org/v2/data/obs/geo/recent"
params = {
    "lat": "{{lat}}",
    "lng": "{{lng}}",
    "dist": "{{dist}}",
    "back": "{{back}}",
    "maxResults": "{{maxResults}}",
    "sort": "{{sort}}",
    "hotspot": "{{hotspot}}",
    "includeProvisional": "{{includeProvisional}}",
    "sppLocale": "{{sppLocale}}"
}
headers = {"X-eBirdApiToken": "{{x-ebirdapitoken}}"}

response = requests.get(url, headers=headers, params=params)
print(response.json())

    const url = new URL("https://api.ebird.org/v2/data/obs/geo/recent");
const params = {
    lat: "{{lat}}",
    lng: "{{lng}}",
    dist: "{{dist}}",

};

Object.keys(params).forEach(key => url.searchParams.append(key, params[key]));

fetch(url, {
    method: "GET",

})
.then(response => response.json())
.then(data => console.log(data));

    üê¶ GET View Checklist

    Fetch the details and observations associated with a specific checklist.

    üîó Endpoint

    API Endpoint

    Base URL: https://api.ebird.org
    Endpoint: /v2/product/checklist/view/{{subId}}
    Method: GET
    Authentication: API token required (X-eBirdApiToken)

    üõ†Ô∏è URL Parameters

    Parameter Description
    subId The checklist identifier.

    üíª Example Requests

    curl --location -g 'https://api.ebird.org/v2/product/checklist/view/{{subId}}' \
--header 'X-eBirdApiToken: {{x-ebirdapitoken}}'
